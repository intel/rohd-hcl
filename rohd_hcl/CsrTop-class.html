CsrTop

CsrTop class

Top level module encapsulating groups of CSRs.

This top module can include arbitrarily many CSR blocks. Individual blocks are addressable using some number of MSBs of the incoming address and registers within the given block are addressable using the remaining LSBs of the incoming address.
