Generate a random FloatingPointValue, supplying random seed rv.

This generates a valid FloatingPointValue anywhere in the range it can represent:a general FloatingPointValue has a mantissa in [0,2) with 0 <= exponent <= maxExponent().

If normal is true, This routine will only generate mantissas in the range of [1,2) and minExponent() <= exponent <= maxExponent().
